Installing an SCP: 5K server with SteamCMD

  • Install SteamCMD

    First install SteamCMD from this link.

    On this page, follow the install instructions depending on your OS. For Windows, you need to click the [1] hyperlink to download SteamCMD.
  • Launch SteamCMD
  • Set install directory

    If you wish to set a custom install directory, use the command:

    force_install_dir (directory)

    Example: force_install_dir C:/SCPServers
  • Login as anonymous

    Login as anonymous by executing the command:

    login anonymous

  • Install SCP Pandemic Dedicated Server

    Install the server by using the command:

    app_update 884110 validate

    This is the same process to update the server when the game gets updated
  • Done

    The SCP 5K server will now install at the designated directory. Once it's done, you can close SteamCMD and go back to the previous page to create the server.
